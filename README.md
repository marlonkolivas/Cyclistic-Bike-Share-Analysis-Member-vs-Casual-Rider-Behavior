## Cyclistic Bike-Share Analysis: Member vs Casual Behavior

### Overview
This project analyzes Cyclistic bike-share data to understand how annual members and casual riders use the service differently. The goal is to identify patterns in user behavior and provide recommendations to increase membership conversion.


### Business Question

How do annual members and casual riders use Cyclistic bikes differently?


###  Tools
- **SQL (MySQL)** — data cleaning and transformation
- **Tableau** — data visualization and dashboard


### Data Preparation
- Converted raw text data into proper formats
- Created key features (ride duration, day, month, hour)
- Removed invalid records (missing values, unrealistic durations)
- Standardized missing station data


### Key Insights
- Members ride more frequently and follow weekday commuting patterns
- Casual riders take longer, leisure-oriented trips
- Casual usage peaks on weekends and during summer months
- Electric bikes are popular across both rider types


### Recommendations
- Target casual riders with seasonal and weekend promotions
- Position memberships as a convenient commuting solution
- Expand and promote electric bike usage


### Dashboard

[Tableau Dashboard](https://public.tableau.com/app/profile/marlon.kolivas/viz/cyclistic_bike_ride_analysis/cyclistic_bike_share_analysis)


### Summary

Members use Cyclistic bikes for consistent, routine transportation, while casual riders use them for flexible, recreational trips. These insights highlight clear opportunities to convert casual riders into long-term members.
